HTML Head

It is a container for metadata such as page title, character set, etc. HTML metadata is data about the HTML document, Metadata is not displayed.

HTML Layout

HTML layout refers to the organizational structure of a web page. It defines how different elements like headings, paragraphs, images, navigation menus, and forms are arranged and positioned on the page.

HTML has several semantic elements that define the different parts of a web page: `<section>`, `<article>`, `<header>` etc

There are four different techniques for HTML Layout:

  • CSS Framework
  • CSS Float
  • CSS Flexbox
  • CSS Grid

HTML Semantics

HTML Semantics refer to using elements that define the meaning and purpose of your content, rather than just its appearance. Examples of semantic element include: `<section>`, `<article>`, `<header>`, etc

HTML Style Guide

It refers to guidelines or tips for creating Good HTML documents/code. Some of them include:

  • Using lowercase for HTML elements naming.
  • Closing all HTML elements.
  • Always quote attribute values.
  • Always specify the alt attribute for img element.
