I. Html Structure

This helps us understand the component need to build web pages with their content. In order to build a web page that fit the standard, there are some component or html element that need to be present to have a well define html document.

The figure above give us an illustration of what html documents (web pages) looks like. We can observe some html elements such as <html> tag, <h1> tag, <head> tag, <body> tag etc. All of those elements and much more are needed in order to build web pages or websites, know let's see in details the meaning to those various tags;

<!DOCTYPE html>
This is used to specify or define the document's type and for this one, it says that the document is an HTML5 Document. Ok Let's go in deeper, we may ask ourselves what is the need of doing this. To answer the question, we need to look at the evolution of the web especial its markup language which is html. HTML has been evolving from type to type with each type having it own characteristics, features & are all use to create websites.
  • XHTML which is a more strict version of html (more sensitive).
  • HTML4 which is a more strict version of html (more sensitive).
  • HTML5 which is an html version that have element with more semantic meaning which will be discuss later.
<html>
This tag is the root of the html document
<head>
This is used to contain metadata or information about the the html document for the browser to better understand the document and helps in SEO optimization via its content. It contains information about the document page title, author, description, link to external resources just to name a few.
<meta>
The HTML tag is use to provide metadata about the HTML document to the browser such the document author, character set used, keywords for search engines, description, refreshing time interval and more. These information are all needed for better SEO optimizations.
<title>
Web pages are usually identified or search by their title or name and this html element is responsible for defining text as the title page for all web pages.
<body>
This is usually define as the content container for all pages. This is due to the fact that all the content display by the browser is written in this section.
<h1>
This is used to define headings in html documents and there exist different headings each one having its own font size and font weight.
