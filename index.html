I. HTML Structure

Understanding the essential building blocks of web pages is crucial for creating websites that adhere to standards. Here are the key components:

The figure above illustrates the basic structure of an HTML document, including elements like `<html>`, `<h1>`, `<head>`, and `<body>`. These and other elements are essential for building well-defined web pages.

<!DOCTYPE html>
This declaration specifies the document type as HTML5, providing information to browsers for accurate rendering.
<html>
This tag defines the root element of the HTML document.
<head>
This section contains metadata about the document, including title, author, description, and links to external resources, all crucial for search engine optimization (SEO) and browser understanding.
<meta>
This tag provides various metadata about the document, such as author, character set, keywords, description, and more, contributing to SEO.
<title>
This element defines the title of the web page, displayed in the browser tab and search results.
<body>
This section acts as the container for all visible content on the page, including text, images, and other elements.
<h1>
This tag defines headings within the document, with different levels (h1-h6) for varying sizes and importance.

IV. HTML Comments

HTML Comments are not display in the browser and are use in documenting the HTML source in-order to ease collaboration among developers.

V. HTML Styles

HTML offers various ways of adding styles to HTML elements, there by providing a better layout to the document.

Inline styles
There are used to apply a unique style to a single HTML element.
Internal styles
There are used to define styles for a single HTML Page.
External styles
There are used to define styles for a multiple HTML Pages.

VI. HTML Formatting

These are HTML elements (`<b>`, `<strong>`, `<i>`, etc) that help us organize and shape text content, making web pages more readable and engaging.

VII. HTML Links

HTML links are hyperlinks that help us navigate through a web page or across many web pages. The HTML `<a>` defines the hyperlink and has the following syntax:

The href is the most important `<a>` attribute and indicates the link's destination (url). HTML Links can be Relative or Absolute.

Relative URLs
They are used for local addressing (linking) i.e link to the a page within the same website, and doesn't contain the `https://` part.
Absolute URLs
They are used for web addressing (external) resources, hence provide the full website or element address.
